Game-Based Learning Approach Using RaspBerry Pi Cyberdeck PAC-MAN For ICT Students in Selected National High-School Capstone. Thesis Project | Co-developed with Engr. Sarlatan and Engr. Castillo.

This project introduced an interactive, game-based learning platform designed to enhance cybersecurity awareness among ICT students using a modified version of the classic PAC-MAN game. Built on a Raspberry Pi-based Cyberdeck, the system simulates real-world malware threats and defense mechanisms within a gamified environment. The solution integrates principles of ethical hacking, system vulnerabilities, and defensive strategies, providing students with an engaging and immersive method to learn core cybersecurity concepts. The implementation was piloted across selected national high schools and demonstrated measurable improvements in learner engagement, knowledge retention, and practical cybersecurity skills.

Co-Developer: Real-Time Structural Health Monitoring System. In collaboration with Engr. Sabesaje and Engr. Callejo

This project involved the development of a real-time monitoring system designed to assess the structural integrity and safety of buildings. Using a combination of sensor networks and microcontroller-based modules, the system continuously tracks key structural parameters such as vibration, tilt, and displacement. Data is transmitted to a locally hosted web interface, allowing users to access building health reports and safety status in real time. The system aims to enhance disaster preparedness, enable early detection of potential structural failures, and support campus infrastructure management through accessible, data-driven insights.

AI-Powered IP Tracking and Threat Intelligence System. Solo Project

This ongoing project aims to build an intelligent IP tracking and threat profiling system that uses artificial intelligence to analyze and respond to real-time network activity. The system is capable of identifying, categorizing, and flagging suspicious IP behaviors such as unauthorized access attempts, brute-force attacks, port scanning, and geolocation inconsistencies. By integrating a lightweight machine learning model, the platform continuously adapts to evolving traffic patterns, improving its threat detection accuracy over time. A centralized dashboard interface presents threat heatmaps, IP activity logs, reputation scoring, and real-time incident alerts, giving users a comprehensive view of potential threats in their network. The system is being developed to support autonomous response actions such as automated blacklisting and alert notifications, with future plans to incorporate predictive threat modeling and compatibility with existing security information and event management (SIEM) systems. Designed for scalability and efficiency, this solution targets small to mid-sized networks seeking affordable, AI-driven security intelligence without reliance on cloud-based infrastructure.
